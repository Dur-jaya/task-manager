📝 Task Management System

📌 Description

This is a full-stack Task Management System that allows users to register, log in, and manage their personal tasks.

The project includes:

 Backend API built with **Node.js + TypeScript + Prisma**
 Web frontend built with **Next.js**
Mobile application built with **Flutter**

---

🚀 Features

🔐 Authentication

 User Registration & Login
 JWT Authentication (Access Token & Refresh Token)
 Secure password hashing using bcrypt

 📋 Task Management

Create, Read, Update, Delete tasks
  Toggle task status (Completed / Pending)
  Tasks are user-specific

 🔍 Advanced Features

* Pagination (load tasks in batches)
* Search tasks by title
* Filter tasks by status

 🌐 Web App (Next.js)

* Responsive UI (desktop + mobile)
* Dashboard with task list
* Add, Edit, Delete, Toggle tasks
* Toast notifications

 📱 Mobile App (Flutter)

* Login screen
* Task dashboard using ListView
* Add, Delete, Toggle tasks
* Pull-to-refresh functionality

---

 🛠️ Tech Stack
Backend: Node.js, Express, TypeScript
Database: SQL (Prisma ORM)
Frontend: Next.js
Mobile: Flutter
Authentication: JWT
Styling: CSS / Inline styling

---

⚙️ Setup Instructions

🔹 Backend

```bash
cd backend
npm install
npx ts-node src/index.ts
```

---

🔹 Frontend (Next.js)

```bash
cd frontend
npm install
npm run dev
```

---

🔹 Flutter App

```bash
cd task_app
flutter pub get
flutter run
```
